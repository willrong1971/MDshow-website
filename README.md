# MDshow Official Static Site

This folder contains a standalone static website for MDshow 1.0.

Open `index.html` directly in a browser. No build step or local server is required.

Pages:

- `index.html`: product landing page and help overview
- `privacy.html`: privacy policy page for App Store Connect and public website use

Content sources:

- `docs/README.md`
- `docs/MDshow-V1.0-Release-Notes.md`
- `docs/MDshow-V1.0-更新公告.md`
- `MDshow/Services/HelpContent.swift`

Version rule:

- The public product version shown on the site is `1.0`.
- Historical `V2`, `V3`, `V4`, and `V5` labels are treated as internal iteration lines, not current public release numbers.
