# MDshow Official Static Site

This folder contains a standalone static website for MDshow 5.0.

Open `index.html` directly in a browser. No build step or local server is required.

Pages:

- `index.html`: product landing page and help overview
- `privacy.html`: privacy policy page for App Store Connect and public website use

Content sources:

- `docs/README.md`
- `docs/MDshow-V5.0-项目状态总结-2026-06-19.md`
- `docs/MDshow-V1.0-Release-Notes.md`
- `docs/MDshow-V1.0-更新公告.md`
- `MDshow/Services/HelpContent.swift`

Version rule:

- The public product version shown on the site is `5.0`.
- The `1.0` content remains the historical baseline; earlier `V2`, `V3`, `V4` labels are historical internal iteration lines, not current public release numbers.
