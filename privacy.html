MDshow 5.0

隐私政策

MDshow 是一款本地优先的 Markdown 编辑器。我们尽量让用户的文档、项目资料和 API Key 留在用户自己的设备与所选择的服务提供商之间。

生效日期：2026 年 6 月 20 日

个人数据与文档内容 AI 与第三方服务 API Key 系统权限 联系我们

个人数据与文档内容

MDshow 本身不会在未经用户许可的情况下收集、存储、上传或出售用户的个人数据、Markdown 文档、导入文件、项目资料、写作草稿或导出内容。

用户在 MDshow 中创建、打开、编辑、导入和导出的文件，默认保存在用户选择的本地位置。应用访问这些文件，是为了完成用户主动发起的阅读、编辑、导入、导出、保存和项目管理操作。

AI 与第三方服务

MDshow 的 AI 功能需要用户自行配置第三方服务提供商的 API，例如 OpenAI、Google Gemini、DeepSeek、Qwen 或其他兼容服务。只有当用户主动使用 AI 对话、翻译、格式整理、文档助手、联网研究或研究写作智能体等功能时，相关文本才可能被发送到用户配置的第三方服务。

这些第三方服务如何处理数据，取决于用户所选择服务的隐私政策、数据处理条款和账号设置。MDshow 不控制第三方服务的数据处理方式，用户应在使用前阅读对应服务提供商的政策。

API Key

MDshow 不提供集中式 AI 中转服务，也不会把用户配置的 API Key 上传到 MDshow 自有服务器。API Key 用于从用户设备向用户选择的第三方 AI 服务发起请求。

请用户妥善保管自己的 API Key。如果怀疑 Key 泄露，应立即到对应第三方服务后台撤销或重新生成 Key。

系统权限说明

  • 文件访问权限：用于打开、读取、编辑、保存 Markdown 文档，导入 Word、Excel、PowerPoint、PDF、网页材料，以及导出 Word、PDF、HTML、Excel、PowerPoint 等文件。
  • 麦克风权限：仅在用户主动使用语音输入或语音转写功能时使用，用于把用户的语音转换为可编辑文本。
  • 网络访问：用于用户主动配置的 AI 服务请求、网页导入、联网搜索、来源深读或其他需要联网的功能。
